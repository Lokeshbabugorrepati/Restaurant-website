# Restaurant Booking - Frontend

React frontend for the Restaurant Table Booking application.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Features

- Beautiful responsive UI
- Real-time booking form
- Booking list with filters
- Status management
- Toast notifications
- Smooth animations

## Technologies

- React 18
- Vite
- Axios
- React Toastify
- CSS3
